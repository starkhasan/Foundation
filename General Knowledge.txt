The World Wide Web commonly known as Web is an information system where documents and other web resources are identified by Uniform Resource Locators (URLs such as http://example

								The resource of the web are transferred via the Hypertext Trasfer Protocol may be accessed by the user by a

An Internet service provider (ISP) is an organization that provides a myriad of service for accessing , using or participating in the internet.
									 An ISP typically serves as the access point or the gateway that provides a user to everything
available on the internet.
