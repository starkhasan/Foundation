Data Warehousing & Data Mining

Data Warehousing includes the process of aggregating data from various database source into one place for efficient access and analysis. Another aspect of data warehousing it the architecture of the data-- that is , how it's structure so that it can be joined, even if the sources have different fields and schema
           Data Mining , is the process of extracting value from the data stored in the data warehouse. Data mining includes the process of transforming raw data sources into a consistent schema to facilitate analysis; identifying patterns in a given dataset, and creating visualization that communicate the most critical insight.
