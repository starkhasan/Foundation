Java Example


		public class demo
                {
                     public static void main(String [] args) // static public void main(String [] args) // public static void main(String args [])
                      {
                        System.out.println(" Hello,World ! ... ");
                      }
                }

Explanation

  • Line begins with the main() methods
  • public, keyword is an access modifier, which allows the programmer to control the visibility of class member
  • static, keyword allows main() to be called without having to instantiate a particular instance of data
  • in main() there is only one parameter provides access to the system
  • System is predefine class that provides access to the system
  • out is the output stream that is connected to the console
