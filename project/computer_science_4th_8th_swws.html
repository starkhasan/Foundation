Semantic Web and Web Services

The Semantic Web is a mesh of data that are associate in such a way that they can easily be processed by machines instead of human operators. It can be concived as an extended version of the existing World Wide Web, and it represents an effective means of data inclusion of semantics contents in Web pages, the semantic web targets the conversion of the presently available Web of understand documents to a Web of information/data.

Semantic Web and Web Services Syllabus

  • Unit-1

    • Introduction to Semantic Web: History of Semantic Web, goals and vision, problems, Semantic Web Technologies, Layered Approach, Syntactic vs semantic web, Applications of semantic web.

  • Unit-2

    • Architecture: XML with Document Type Definitions and Schema, addressing and quering XML documents, RDF (Resource Despription Framework),basic idea and syntax, quering in RQL, URL

  • Unit-3

    • Ontologies: Role of Ontology in intelligent information retrieval on web, OWL, Ontologies for different application. Ontology engineering: construction ontologies manually, reusing existing ontologies.

  • Unit-4

    • Semantics:Kinds of semantics, use of semantics, Search Engines: Role of search Engines in intelligent retrieval of information on Web, Semantic web browser.

  • Unit-5

    • Logic and inference: example of Monotonic rules: family relationships, monotonis relu:syntax and semantics, Non-monotonic and Non-Monotonic rules.
